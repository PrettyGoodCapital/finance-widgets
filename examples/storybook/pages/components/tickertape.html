Ticker Tape

This library is in pre-alpha stage and is not yet suitable for production usage.

A scrolling ticker-tape component. Displays a tuple of (string, number1, arrow, number2), where arrow will be up if number2 > 0, flat if number2 == 0, otherwise down. The string will show as bold, while the colors of the arrows are controlled by their corresponding attributes. Note that the attributes tickers, prices, and changes should all be arrays of the same length.

Types

  • QuoteMiniData

Properties

  • data Array<QuoteMiniData> The data.
  • delay Number Controls the scroll speed of the tickertape, leave blank for automatic calculation based on the number of elements.
  • color_positive String Controls the color of the positive indicator, defaults to "var(--sl-color-success-500)"
  • color_negative String Controls the color of the positive indicator, defaults to "var(--sl-color-danger-500)"
  • color_neutral String Controls the color of the positive indicator, defaults to "var(--sl-color-neutral-500)"
