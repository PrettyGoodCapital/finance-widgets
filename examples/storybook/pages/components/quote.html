Quote

This library is in pre-alpha stage and is not yet suitable for production usage.

A single asset quote widget.

Types

  • QuoteData
    • ticker string
    • name string
    • market string
    • price number
    • change number

Properties

  • data QuoteData The data.
  • orient String Controls whether the change is displayed below ("vertical") or next-to ("horizontal") the price. Default is "horizontal".
  • color_positive String Controls the color of the change text when positive, defaults to "var(--sl-color-success-500)"
  • color_negative String Controls the color of the change text when negative, defaults to "var(--sl-color-danger-500)"
  • color_neutral String Controls the color of the change text when flat, defaults to "var(--sl-color-neutral-500)"

Methods

  • updateData(newPrice, newChange)
    • newPrice number
    • newChange number
