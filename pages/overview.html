Overview

finance-widgets is a library of simple but extensible UI widgets for financial applications. Each widget is designed as a standalone WebComponent, and the libary relies on Lit and Shoelace for the bulk of the UI heavy lifting.

This library is in pre-alpha stage and is not yet suitable for production usage.

Installation

Data Models

Data models represent data types that are required for widgets to function. This data is JSON based, with typescript typings automatically generated from JSON Schema. Providers can be instantiated to automatically bind data providers to collections of widgets (more on this below).

npm i @finance-widgets/core

UI Components

Core widgets, available as WebComponents. Browse the available widgets on the left side bar of this page.

npm i @finance-widgets/core-ui

Providers

Providers bind data models to widgets. These are designed to be implemented by the end user for their own data sources, but a few are provided.

npm i @finance-widgets/provider
