Code Usage

This library is in pre-alpha stage and is not yet suitable for production usage.

A component to render a description, block of code, and an example of that code. Uses the fw-utils-codeblock component.

Attributes

  • language String The language for which to render.
  • code String The code to render.

Slots

  • description Rendered in the top panel.
  • example Rendered in the bottom panel.
